# HR-Karavan Swarm — Совет Биев

> AI-модуль для оценки и генерации целей сотрудников по методологии SMART с многоагентным подходом

## 🎯 О проекте

Система автоматизирует процесс постановки и оценки целей сотрудников. Вместо одного "чёрного ящика" мы используем **Совет Биев** — 5 независимых агентов, каждый оценивает свой критерий SMART.

### Ключевые возможности

- ✅ **SMART-оценка** — анализ цели по 5 критериям (Specific, Measurable, Achievable, Relevant, Time-bound) с пояснениями
- ✅ **Генерация целей** — предложение 3-5 целей на основе должности и подразделения
- ✅ **Дашборд руководителя** — визуализация зрелости команды, выявление проблемных зон
- ✅ **Explainable AI** — каждый балл обоснован, никаких "чёрных ящиков"
